# Deutsches Modul — PoemReason

## Dateien

- **`g2p.pl`** — Vereinfachte Phonem-Graphem-Umsetzung für das Deutsche.
  Behandelt Umlaute, ß, Diphthonge (ei, au, eu), sch/ch, Dehnungs-h,
  Auslautverhärtung.
- **`phonetics.pl`** — Silbenzählung (akzentuell-silbisch), keine
  Betonungs-Kürzung. Tradition: deutsch_silabico.

## Poetische Tradition

Die deutsche Metrik ist akzentuell-silbisch:
- Der Knittelvers (Goethe, Faust) — 4 Hebungen, Paarreim
- Der Blankvers — reimloser jambischer Fünfheber (10-11 Silben)
- Die Volksliedstrophe — 4-zeilig, 4/3 Hebungen im Wechsel
- Der Alexandriner — 12-13 Silben, AABB
- Das Distichon — Hexameter + Pentameter (Goethe, Schiller)

## Lokale Formen

- **knittelvers** — Paarreim, freie Füllung, AABB, 4-er Blöcke
- **blankvers** — 10 Silben, reimlos
- **volksliedstrophe** — 8-6-8-6, ABAB
- **alexandriner** — 12 Silben, AABB
- **distichon** — Hexameter (16) + Pentameter (14), reimlos

## Einschränkungen

- Keine Unterscheidung /x/ vs /ç/ für 'ch'
- Vokallänge nicht markiert
- Diphthonge approximiert
- Keine Kompositumsbetonung
